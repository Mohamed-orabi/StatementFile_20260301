MSCC have 2 main types of statements 

1 – Text file (no charge)
Text (Spool) file for Printer Xerox (Processed by Xerox) not suitable to print by end user and must processed by Xerox service center

2 – PDF format contain 2 types (with charge)
- PDF preprinted that will print on template of statement designed by bank , we will send the bank PDF file will print on preprinted paper, it mean the bank will prepare paper contain statement layout, logo and labels then MSCC send PDF file contain only values printed on that layout
- PDF file will print by bank on blank papers. The file will print statement layout, logo, labels, and values on white paper
